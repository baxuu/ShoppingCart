# ShoppingCart

ShoppingCart is a simple shopping cart management application.

![]()

### Built with

- `React,`
- `Redux,`


## Setup

To run this project, install it locally using npm:

```
$ cd ShoppingCart
$ npm install
$ npm start
```

To launch the test runner:

```
$ cd ShoppingCart
$ npm test
```
